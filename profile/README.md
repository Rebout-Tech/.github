# Reboot-Tech 👋

Здесь публикую готовые, самостоятельные продукты и сервисы. В отличие от библиотек, это законченные решения, которые можно использовать "из коробки".

## 🌐 Ресурсы

- **Сайт:** [rebout.ru](https://rebout.ru)

## 🛠 Технологический стек

- **Backend:** C#, Elixir
- **Frontend:** TypeScript
- **Инфраструктура:** Docker, RabbitMQ, PostgreSQL

## 📦 Проекты

- **Volucio** — S3-совместимое хранилище на Elixir/Phoenix
- **NotifyX** — сервис уведомлений на Elixir
